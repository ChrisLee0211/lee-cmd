# How to use?
```
cargo build --release
```
then
```
cargo install --path .
```

# feature
- git push origin xxx
```
lee ph
```
- git pull origin xxx
```
lee pl
```
- git add & commit & push origin branch
```
lee cph -m "xxxx"
```
- copy current branch name
```
lee br
```
- remove cargo cache
```
lee rc
```

# How to test
```
./target/debug/lee {your command}
```
